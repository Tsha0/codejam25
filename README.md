# CodeJam 2025 - Creative Battles

A real-time competitive coding game where players compete by creating web designs based on prompts.

### Backend

```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

### Frontend

```bash
cd client
npm install
npm run dev
# Runs on your network, port 3000
```

## Tech Stack

**Backend:**
- Python 3.12+
- Flask + Flask-SocketIO
- MongoDB

**Frontend:**
- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
